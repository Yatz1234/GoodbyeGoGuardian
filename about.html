Goodbye, GoGuardian!

Say goodbye to the most annoying privacy destroyer on the internet!

About

This project is a collaboration between @elipie and @N3rdL0rd on replit. It allows students to have free access to the internet and internet resources without being tracked, monitored, and censored by teachers. This was a project made by students, for students. Please use this project responsibly.

NOTICE:
Once you leave the GoodbyeGoGuardian window, if you go back to the tab, the title of the window WILL show up on the teacher's GoGuardian monitor. If you accidentally minimize your window, you MUST relaunch it unless you would like to use it explicitly.
